GnuCash/Quote.pm
Last night I grabbed the CVS sources of GnuCash, looking for a better way of keeping track of my finances. In a fit of hacking I added support for stock quotes from the Australian Stock Exchange (ASX), and posted patches back to SourceForge

Unfortuately, SourceForge ate my patch -- or more correctly, ate all the backslashses in my patch. I don't seem to be the only only afflicted by this problem, a few of the other patch-files I looked at seemed to have a large number of strings ending with "...n". If any readers would like a copy of the (un-munged) patches, please let me know.

Contacted the other authors of the Quote.pm module regarding a release to CPAN. I would have loved a module like this about a month ago when I was toying with stock-monitoring software.
